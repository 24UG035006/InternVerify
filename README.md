# InternVerify

InternVerify is a Flask-based Internship Verification & Risk Assessment Platform.

## Features

- Risk Score Calculation
- Trust Score Analysis
- Scam Probability Detection
- Recommendation Engine
- Safety Tips
- Internship History Tracking
- Company Search
- Dashboard Analytics
- Pie Chart Visualization
- PDF Report Download
- Excel Report Download
- Dark Mode Support

## Technologies Used

- Python
- Flask
- HTML
- CSS
- SQLite
- Matplotlib
- ReportLab
- OpenPyXL

## How to Run

```bash
pip install flask matplotlib reportlab openpyxl
python app.py
